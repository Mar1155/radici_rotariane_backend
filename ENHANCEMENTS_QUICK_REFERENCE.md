# Quick Reference: Enhanced seed_cards.py

## What Changed?

### 1. Cover Images Implementation
Every card that requires `cover_image` in its section configuration now gets a random image from picsum.photos.

**Coverage by Section:**
```
✅ Storie e Radici (storie & tradizioni tabs):        6 cards with images
✅ Calendario Radici:                                  4 cards with images  
✅ Scopri la Calabria (itinerari & esperienze):      8 cards with images
⚪ Other sections:                                      No images (not in config)
```

**How It Works:**
```python
# Each call increments counter by 1-100 for uniqueness
self._get_random_image_url()
# Returns: https://picsum.photos/300/200?random=8792
```

### 2. Rich HTML Content
All 44 cards now use a new `RichContentGenerator` class that creates professional HTML with:

**Formatting Features Available:**
- `<h1>` and `<h2>` headings
- `<strong>` bold, `<em>` italic, `<u>` underline
- Colored text (`<span style="color: #color">`)
- Highlighted text (`<span style="background-color: #color">`)
- `<ol>` ordered lists
- `<p style="text-align: center">` centered text
- `<a href="">` links to resources

**Example Output:**
```html
<h1>Un itinerario di 3 giorni nel cuore del Parco Nazionale</h1>
<h2>Dettagli Importanti</h2>
<p>Escursione mattutina al Lago Cecita, uno dei laghi più suggestivi d'Italia...</p>
<ol>
  <li>Trekking tra il bosco di Carlomagno</li>
  <li>Visita al borgo di San Giovanni in Fiore</li>
  <li>Salita al Botte Donato (1928m)</li>
</ol>
<p><a href="https://www.rotary.org" target="_blank">Learn more at Rotary.org</a></p>
```
